What Is Solid Now

Math and Engine Guardrails

Amortized payoff helpers, LOC average-daily-balance interest, daily interest burn, non-finite input sanitization, horizon-exceeded states, over-limit states, full-LOC no-capacity states, and modeled interest-difference labels are covered by regression tests.

Dashboard and Money Loop

The dashboard has the required four vitals, a model-backed Money Loop artifact rail, selected-token depth layers, payoff orbit visual, change explanations, compact mobile vitals, and mobile Money Loop bridge.

Truth-First Copy

Guardian, Learn, Simulator, Cockpit, Portfolio, Vault, and Dashboard copy now consistently label assumptions, avoid guaranteed savings language, and keep LOC routing conditional on real terms, fees, timing, and cash-flow recovery.

Portfolio and Simulator Semantics

Portfolio Velocity is framed as a planning default, while Simulator comparison explains fastest/least-interest analysis separately. Invalid projections show review states instead of claiming payoff wins.

Backend Transparency

Settings exposes local demo storage, backend handoff snapshot export/import, provider-neutral decision gates, Supabase first-lane schema handoff, and Cloudflare edge-lane documentation without pretending live persistence is wired.

Mobile and Expo Foundation

Expo app shell exists with dashboard, simulator, cockpit, portfolio, learn, vault, and settings routes. Expo web export and mobile contract tests are part of CI. Native device smoke remains environment-dependent.
