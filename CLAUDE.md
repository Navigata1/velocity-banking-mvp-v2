# CLAUDE.md — Working Agreement for Velocity Banking App

## Product truth
This repo builds a calm, empowering simulator + dashboard for the “Money Loop.”
It must be a truth machine: clear assumptions, real math, no hype.

## Non-negotiables
- Offline-first
- Privacy-first
- Guardrails for negative cash flow and overutilization
- Plain-language UI

## How to contribute
1. Read /docs/01_NORTH_STAR.md
2. Pick a ticket (see /docs/06_SPRINT_IMPLEMENTATION_GUIDE.md)
3. Write tests for engine changes
4. Demo in a simulator scenario

## Definition of Done
- Works on iOS + Android
- Passes engine unit tests
- Handles empty states
- Copy is calm and non-shaming
